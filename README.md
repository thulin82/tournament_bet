* [PHP](http://php.net/) - The latest version of PHP is highly recommended
* [Composer](https://getcomposer.org/) - Dependency Manager


## Install
#### Dependencies
```CMD
make composer-install
```
#### Run database migrations and seed with test data
```CMD
make database
```
